Target systems (beyond current prototype)

  • Physics: grip vs drift blend, surface types (asphalt / oil / boost pad), slight suspension squash, wall scrape sparks, speed-sensitive FOV/zoom.
  • Cars: 5 unique chassis (speed / armor / handling / damage). Player unlocks skins later; AI roles use different weights.
  • Maps: Figure-8 chaos, oval bowl, city block, canyon run — shared Track API, per-map Path2D + power-up nodes.
  • Weapons: always-on MG + missile; pickups for oil, shield, nitro, EMP. Hotbar on HUD (mockup above).
  • UI: glass chips, gold CTA, mode pills, lap progress bar, weapon slots, result pop-in (partially already in prototype).
